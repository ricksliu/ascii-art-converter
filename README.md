# ascii-art-converter

Converts videos into ASCII art. Uses Python with opencv, pillow and moviepy.

## Setup

1. Place videos in `input/`. Various settings can be changed in `config.json`.

2. Run `main.bat`. Converted videos will appear in `output/`.

# ascii-art-converter

Converts videos into ASCII art. Made with Python, cv2, PIL and moviepy.

## Instructions

1. Place videos in `input/`. Various settings can be changed in `config.json`.

2. Run `main.py`. Converted videos will appear in `output/`.
